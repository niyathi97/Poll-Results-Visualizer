# 📊 Poll Results Visualizer

## 🚀 Overview

The **Poll Results Visualizer** is a data analytics project that transforms raw survey data into meaningful insights using interactive visualizations and dashboards.
This project analyzes real-world survey data and presents trends related to user responses, demographics, and behavioral patterns.

---

## 🎯 Objective

To build a system that:

* Loads poll/survey data from CSV files
* Cleans and preprocesses raw data
* Performs exploratory data analysis (EDA)
* Generates visual insights using charts
* Displays results through an interactive Streamlit dashboard

---

## 🧠 Problem Statement

Survey and poll data are often large and difficult to interpret manually.
This project simplifies analysis by converting raw responses into clear visual insights, enabling faster and better decision-making.

---

## 💡 Features

* 📥 Data loading from CSV files
* 🧹 Data cleaning and preprocessing
* 📊 Exploratory Data Analysis (EDA)
* 📈 Interactive charts using Plotly
* 🎛️ Dashboard filters (Country, Gender)
* 📌 KPI metrics (Total responses, Avg age, Countries)
* 🌍 Country-wise analysis
* 👥 Gender distribution
* 🎯 Age distribution
* 🧠 Treatment vs No-treatment analysis
* 💼 Work interference insights

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```
git clone https://github.com/yourusername/Poll-Results-Visualizer.git
cd Poll-Results-Visualizer
```

### 2️⃣ Create Virtual Environment

```
python -m venv venv
venv\Scripts\activate
```

### 3️⃣ Install Dependencies

```
pip install -r requirements.txt
```

---

## ▶️ Run Project

### Run Analysis Script

```
python main.py
```

### Run Dashboard

```
streamlit run app.py
```
